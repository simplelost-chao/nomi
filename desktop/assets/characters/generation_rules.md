# 角色图片生成规则

每次生成角色表情图片前必须遵循以下原则：

## 1. 风格匹配
- 必须尽量模仿参考图的画风和风格
- 如果参考图是写实风格就用写实，参考图是Q版就用Q版
- 不要自作主张改变风格

## 2. 背景透明（真正的 alpha 通道透明）
- 所有图片必须有**真正的 alpha 通道透明背景**
- 不是画棋盘格假装透明，不是纯黑，不是纯白
- prompt 中写 "transparent background, PNG with alpha channel"
- 生成后**必须**使用 rembg 库做专业背景去除（不要用简单的颜色阈值）
- 验证标准：角落像素的 alpha 值必须为 0

## 3. 人物一致性
- 所有 7 个表情状态的人物必须保持一致：
  - 体型比例一致（头身比固定）
  - 身高一致
  - 服装一致
  - 发型发色一致
- 图片尺寸统一

## 4. 全身像
- 每张图必须是完整的全身像
- 头、手、脚都必须完整可见
- 不要只有半身或头像

## 5. 图片规格
- 输出为 PNG 格式
- 透明背景
- 生成后裁剪到内容边界 (crop to content bounds)
- 所有状态图的人物大小应该接近一致
